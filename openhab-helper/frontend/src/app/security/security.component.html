Security Settings

Adjust the way how SmartBerry secures your home.

Operation State

The security system protects your home and responds to any unauthorized activity. If openHAB-Cloud and the operation state is set up properly, SmartBerry notifies you as soon as smoke or an assault had been detected. Adjust the operation state by editing Core_Security_OperationState. Current choices are:

  • No notifications and assault alarms (OFF)
  • Notifications only (SILENTLY)
  • Notifications and assault alarms (ON)

You may let items disable the operation state automatically. Assault disarmer items turn off the operation state, if the item state changed to ON or OPEN.

Assault Detection

Tag your doors or windows as assault trigger items to enable assault detection. Any state change of these items will fire an assault event during an active operation state.

To enable assault alarms right after an assault detection, tag the corresponding siren or security system as assault alarm item. You can change the duration until the alarm will turn off automatically by editing the Core_Security_AssaultAlarmAutoOff item. Set to zero to disable the auto-off functionality.
